# Генерация токена Yandex.Market

Для генерации токена, в [Личном кабинет партнера Yandex.Market](https://partner.market.yandex.ru/) :
1. В левом нижнем углу нажмите на название своего бизнеса и перейдите на страницу Модули и API.
2. В блоке Авторизационный токен нажмите кнопку Создать или Создать новый токен, если вы уже получали его.
3. В открывшемся окне указать уникальное имя токена, выбрать доступы, нажать кнопку Создать.


Для корректной работы, токен должен иметь доступы:
- Общение с покупателями
- Обработка заказов и учёт товаров
- Просмотр товаров и карточек
- Просмотр цен
- Настройка магазина

<details>
    <summary>Для чего нужны права доступа</summary>

    - для получения информации о габаритах товаров для накладной.
    - для получения информации о заказах.
    - для загрузки объявленной стоимости товара в накладную.
    — для отправки кода получения покупателям.
Важно! Токен не должен быть тестовым, иначе интеграция не сможет передавать данные о заказах между системами.

</details>

Необходимо скопировать сгенерированный токен и вставить его в [настройки интеграции](https://lk.cdek.ru/integration).

:::warning

Токен будет доступен только один раз после создания. При его утере, необходимо будет создать новый.

:::

После заполнения поля, необходимо вернуться в [личный кабинет СДЭК](https://lk.cdek.ru), подтвердить ознакомление с
согласием на передачу данных и отправить токен на проверку, нажав **Проверить API-ключ**.

<Image img={require('./img/yandex-key/key.png')} />

<details>
    <summary>Сколько занимает проверка токена?</summary>

    <Image img={require('./img/yandex-key/check.png')} />

    Проверка токена занимает от 1 минуты до нескольких часов, в зависимости от нагрузки на сервер интеграции.

    Если проверка занимает более 24 часов, необходимо обратиться в [техническую поддержку СДЭК](mailto:integrator@cdek.ru).

</details>

<details>
    <summary>Ошибка проверки токена</summary>

    <Image img={require('./img/yandex-key/error.png')} />

    В случае ошибки проверки токена, необходимо проверить правильность введенного токена и выданных ему прав доступа.
    При необходимости, требуется создать новый токен и повторить процедуру.

    Если проблема не решается, необходимо обратиться в [техническую поддержку СДЭК](mailto:integrator@cdek.ru)

</details>

В случае успешной проверки API-ключа, будет создан новый интегрированный магазин и станет доступна форма настройки
интеграции.
